# Deq

[Deque](https://www.concatenative.org/wiki/view/Deque)-inspired interpreted programming language.

## Quick start

### Build

Dependencies:
- C++ compiler (Preferably GCC or Clang. MSVC might not work)
- Make

```console
$ make
```

### Usage

- [Examples](./examples/)

```console
$ ./deq file.deq
```
